# Proyecto: Servidor Web en la Nube con Monitoreo

Este proyecto consiste en la creación de una **máquina virtual en la nube** para desplegar un servidor web con **Apache, PHP y MySQL**, acompañado de un **sistema de monitoreo básico** para CPU, RAM, Disco y conectividad.

## 🚀 Tecnologías utilizadas
- Google Cloud Platform (Compute Engine)
- Apache 2
- PHP 7.4+
- MySQL / MariaDB
- Linux Ubuntu 20.04
- Herramientas de monitoreo (`htop`, `df -h`, `ping`, revisión de logs)

## 📋 Pasos principales
1. Creación de la máquina virtual en la nube.  
2. Instalación de Apache, PHP y MySQL.  
3. Configuración de un sitio web básico.  
4. Monitoreo de recursos de la VM.  
5. Generación de reportes en Excel.  

## 📊 Reportes
Los reportes de monitoreo incluyen:
- Uso de CPU y RAM  
- Espacio en disco  
- Latencia de red (ping)  
- Logs de sistema  

👉 Los reportes están disponibles en la carpeta `/reports` en formato **Excel**.

## ✅ Resultados
- Servidor web funcionando en la nube.  
- Monitoreo manual con reportes en Excel.  
- Documentación completa del proceso en PDF.
